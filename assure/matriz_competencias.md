## 3. Databases

### Fundamentals (Software Engineer I)
**Descripción:** Conoce los fundamentos básicos de bases de datos relacionales.

**Conceptos fundamentales:**
- Data Models
- DBMS (Database Management Systems)
- Normalization
- Constraints
- Indexes
- Tables & Data types
- SQL (Structured Query Language)

### Query Languages I
**Descripción:** Conoce sentencias básicas DQL y tiene capacidad de utilizar un DBMS para ejecutar consultas.

- DRL/DQL (Data Retrieval Language/Data Query Language)

### Query Languages II
**Descripción:** Tiene la capacidad de ejecutar lenguajes de subconsulta más avanzados como DCL y SCL.

- DDL (Data Definition Language)
- DCL (Data Control Language)
- SCL (Session Control Language)

### Non-Relational (NoSQL)
**Descripción:** Conoce los conceptos fundamentales de bases de datos no relacionales y es capaz de identificar el tipo apropiado a ser utilizado de acuerdo al contexto.

- Concept
- Attributes of NoSQL Databases
- Advantages / disadvantages
- Data types

### Administration (DBA Basics)
**Descripción:** Conoce las responsabilidades básicas de un DBA al punto que le permita trabajar de forma conjunta sugiriendo mejoras en el diseño de base de datos y optimización de consultas.

- Ability to create/review a db model/script following good practices and normalization rules
- Ability to review performance query issues to suggest a good way to improve the query
- Table Partitioning

### Advanced Topics
**Descripción:** Conoce conceptos avanzados de bases de datos, como ser Big Data, replicación, almacén de datos (Data Warehouse) y minería de datos (Datamining).

- Big data
- Database Replication
- Datawarehouse
- Data mining

### Tuning and Performance
**Descripción:** Entiende de forma clara los beneficios y cuidados en el uso de indices de bases de datos y es capaz de entender, analizar y optimizar un plan de ejecución de una consulta SQL.

- Indexes
- Execution Plan
